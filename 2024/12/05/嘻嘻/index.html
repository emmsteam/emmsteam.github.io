8. 功夫再高也怕菜刀

flag: flag{3OpWdJ-JP6FzK-koCMAK-VkfWBq-75Un2z}

题解

这是一道流量分析题，涉及到 HTTP 过滤和包的分析。具体步骤如下：

  1. 打开抓包工具
    小鲨鱼（抓包工具）开始分析流量，首先要找到正确的包头。此时我们需要不断查看包的内容，直到发现一个特殊的标志位：PK（即包头），这个标志位里藏有 flag。

  2. 导出包
    一旦找到了正确的包头，你需要将数据导出来。这里有几个选择：

    • 可以选择导出分组字节流；
    • 也可以选择原始数据另存为文件。

    注意： 不要选择跟踪 TCP 流。因为这是分段传输的，TCP 流中会有标志位，导致导出的压缩包可能会显示损坏。

  3. 分析返回包
    接下来要做的是只关注 HTTP 返回包。选择原始数据并保存。

  4. 图片隐藏密码
    通过分析这些包，我们发现其中包含了一张图片。你可能会想，为什么要上传一张图片呢？是不是密码就藏在里面？

  5. 提取密码
    把图片导出后，用 Notepad 打开，并将其内容转为 十六进制。接着，你会看到，图片的内容实际上是经过编码的 十六进制码。然后将十六进制码转换为 ASCII 码，最终就能得到密码啦！

通过这些步骤，我们成功地找到了隐藏的 flag 和密码！是不是觉得很有趣呢？嘻嘻~
